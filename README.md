# cloud_gaming_analytics
Анализ активности пользователей облачного гейминг-сервиса на основе данных об игровых сессиях
## Описание проекта

Проект направлен на расчёт ключевых метрик активности пользователей облачного гейминг-сервиса для понимания их поведения и оптимизации продукта.

### Рассчитываемые метрики:

| № | Метрика | Описание |
|---|---------|----------|
| 1 | **DAU** | Число уникальных пользователей в день |
| 2 | **Sessions by Platform** | Количество сессий для каждой платформы (PC/Android/iOS) |
| 3 | **Avg Session Duration** | Средняя длительность сессии по платформам |
| 4 | **New Users Share** | Доля сессий новых пользователей |
| 5 | **Median Session** | Медиана длительности сессий для новых и старых пользователей |
| 6 | **Total Time (RU)** | Суммарное время в игре для пользователей из России |

## Технологии

- **Python 3.8+**
- **Pandas** — обработка и анализ данных
- **SQLite** — хранение и SQL-запросы к данным
- **Jupyter Notebook** — интерактивная разработка

## Ключевые выводы

- Пользователи **iOS** проводят в игре больше времени (в среднем ~47 мин)
- Доля сессий от **новых пользователей** составляет около 30%
- Медианная длительность сессии у новых и старых пользователей практически одинакова (~31-32 мин)
